# Usage
Choose the right branch in this repository to get the code for the different parts of the series.

Git Clone
```sh
git clone https://github.com/nileshdt/expo-react-native.git
```
Install all dependencies
```sh
yarn install
```

Run the server
```sh
yarn start
```
Run on android
```sh
yarn android
```
